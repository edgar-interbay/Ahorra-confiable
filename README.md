Ahorra Confiable es un programa que creamos con la idea de ayudar a las personas a saber cuánto tiempo necesitan ahorrar para poder comprar algo que desean. Muchas veces uno sabe cuánto quiere ahorrar, pero no cómo calcular el tiempo que tomará llegar a esa meta, así que este programa se encarga de hacer ese trabajo por ti.



¿Qué hace este programa?:

Te permite elegir cómo quieres ahorrar: semanal, bisemanal, mensual, etc.

Te pregunta cuánto puedes guardar por cada periodo.

Te deja escoger para qué quieres ahorrar.

Luego calcula, de forma aproximada, cuántos años y meses tardarías en reunir el dinero.





HOW TO USE:

-Abre el programa desde tu computadora.

-Selecciona el tipo de ahorro que prefieres.

-Ingresa cuánto puedes ahorrar en ese periodo.

-Elige el tipo de artículo o meta que quieres alcanzar.

-Ingresa el precio aproximado.

-El programa te mostrará cuánto tiempo tardarías en llegar a esa cantidad.
